简介 什么是模式 基础 JSON Schema手册 Type定义 通用关键字 组合模式 allOf anyOf oneOf not $schema关键字 正则表达式 构建一个复杂的模式

oneOf

oneOf意思是，对于给定的多个schema，只能有一个有效。

// a schema
{
    "oneOf": [
        { "type": "number", "multipleOf": 5 },
        { "type": "number", "multipleOf": 3 }
    ]
}

// success
10

// success
9

// failure
15

上面schema也可以写成下面这样：

// a schema
{
    "type": "number",
    "oneOf": [
        { "multipleOf": 5 },
        { "multipleOf": 3 }
    ]
}
